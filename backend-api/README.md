# Environment setup :
The backend API is built with Java version 21, Spring 3.5 and Maven version 3.9.x.  
In order to run the backend API, you need to have these tools installed on your machine:
> - [JDK 21](https://adoptium.net/en-GB/temurin/releases)
> - [Maven 3.9](https://maven.apache.org/download.cgi)

# How to start the backend api:

Open a terminal, go in the `backend-api` folder and run this command:  
```
mvn spring-boot:run
```

# API Documentation:
Once the backend API is running, you can access the API documentation at this URL:  
```
http://localhost:8080/service/swagger-ui/index.html
```
